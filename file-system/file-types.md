# File types



| Attribute | File type                                                                              |
| :-------: | -------------------------------------------------------------------------------------- |
|     -     | a normal file                                                                          |
|     d     | a directory                                                                            |
|     l     | a symbolic link                                                                        |
|     b     | a special block file. They're files that manage information of blocks data like an USB |
|           |                                                                                        |
