# File compression



```
dmesg                                     -> creates an archived file from a file
gzip [file_name]                          -> compress a file with the .gz extension
tar cf [compressed_file.tar] [file_name]  -> extracts archived file
tar czf [compressed_file.tar.gz]          -> compresses a file with the .gz 
tar -czvf <name> .tar.gz <directory_name> -> compresses a file 
tar xf [compressed_file.tar]              -> creates a gzip compressed tar file
tar -xzvf <name> .tar.gz <directory_name> -> uncompresses a file
```
