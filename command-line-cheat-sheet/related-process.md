# Related process



```
ps                   -> shows a snapshot of active processes
pstree               -> shows processes as a tree
pmap                 -> shows a memory usage map of processes
top                  -> shows all running processes
kill [process_id]    -> kills a process under a given ID
pkill [proc_name]    -> kills a process under the specified name
killall [proc_name]  -> kills all processes labelled proc
bg                   -> lists and resume stopped jobs in the background
fg                   -> brings the most recent suspended job to the foreground
fg [job]             -> brings a particular job to the foreground
lsof                 -> lists files opened by processes
```
