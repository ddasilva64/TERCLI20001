# File permission



```
chmod 777 [file_name]               -> gives read, write, and execute permission to everyone
chmod 755 [file_name]               -> gives full permission to owner, and read and execute permission to group and others
chmod 766 [file_name]               -> gives full permission to owner, and read and write permission to group and others
chown [user] [file_name]            -> changes the file ownership
chown [user]: [group] [file_name]   -> changes the owner and group ownership of a file
```
