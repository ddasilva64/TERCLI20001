# Command exploration and help within the terminal



```
man command                 -> shows the command user manual
help command                -> shows help for the command (only works in bash shell)
which command               -> shows the path where command executable is located
whatis command              -> shows the command function
alias aliasname = ”command” -> creates an alias for the command
alias l = ”ls -la”          -> creates an alias for the ls command with l options
```
